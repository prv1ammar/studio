# Comprehensive List of n8n Nodes (Categorized)

# --- CORE & FLOW CONTROL ---
# Nodes for managing the execution flow of workflows.
[core] Start - The starting point of a workflow.
[core] Schedule - Trigger workflows at specific times or intervals.
[core] Webhook - Trigger workflows via HTTP requests.
[core] Stop and Error - Error handling node.
[flow] If - Conditional logic (boolean true/false).
[flow] Switch - Route data based on values (multi-path).
[flow] Merge - Combine data from multiple branches (Append, Keep Key, etc.).
[flow] Split In Batches - Loop over items in batches.
[flow] Item Lists - Helper to split/aggregate lists of items.
[flow] Wait - Pause the workflow for a duration or until a specific time.
[flow] Compare Datasets - Find differences between two datasets.
[flow] Execute Workflow - Trigger another n8n workflow.
[flow] Set - Set values on items.
[flow] Function - Execute custom JavaScript code (Legacy).
[flow] Function Item - Execute custom JavaScript code per item (Legacy).
[flow] Code - Execute custom JavaScript or Python code (Modern).
